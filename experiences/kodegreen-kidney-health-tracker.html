Kingston University
January - March 2023

Project Overview

Spearheaded the complete full-stack development of the KodeGreen Kidney Health Tracker, a sophisticated web application for monitoring kidney health metrics, personally implementing both frontend interfaces and backend architecture whilst working in a collaborative team environment. The application enabled patients and clinicians to calculate, visualise, and track kidney function through eGFR (estimated Glomerular Filtration Rate) measurements with tailored interfaces for different user roles.

Frontend

  • Chart.js for dynamic data visualisation
  • Responsive design for all devices
  • Role-based user interfaces

Backend

  • RESTful API architecture
  • MySQL database via AWS RDS
  • AWS cloud infrastructure (EC2, Route53, S3)

Security

  • Access control middleware

Development
